SiteVoice

Informativa sulla Privacy

Ultimo aggiornamento: 26 marzo 2025
In sintesi: SiteVoice elabora le tue registrazioni vocali per generare report tecnici di cantiere. L'audio viene inviato a OpenAI per la trascrizione, i report vengono salvati sul tuo dispositivo e, se hai un account, nel cloud Supabase. Non vendiamo, condividiamo né utilizziamo i tuoi dati per scopi pubblicitari.

1. Chi siamo

SiteVoice è un'applicazione mobile sviluppata da Alessandro Bracco (bracco.ale@gmail.com) che consente a professionisti dell'edilizia di creare diari di cantiere e report tecnici tramite registrazioni vocali.

2. Dati che raccogliamo

🎙️ Registrazioni audio Audio registrato tramite il microfono del dispositivo. Viene inviato ai server OpenAI per la trascrizione e non viene mai salvato in modo permanente da noi. L'audio è elaborato in tempo reale e scartato al termine.
📝 Trascrizioni e report Il testo trascritto e formalizzato da OpenAI GPT-4o viene salvato localmente sul tuo dispositivo (AsyncStorage). Se crei un account, i report vengono sincronizzati nel cloud Supabase.
📷 Fotografie Le foto che allegi ai report vengono salvate localmente e, se hai un account, caricate nel cloud Supabase. Non vengono inviate a terze parti per elaborazione.
✍️ Firma digitale Le firme digitali vengono salvate come immagine (base64) all'interno del report, localmente e nel cloud se hai un account.
📧 Indirizzo email Solo se scegli di creare un account. Utilizzato esclusivamente per l'autenticazione tramite magic link. Non viene usato per comunicazioni commerciali.

3. Come utilizziamo i tuoi dati

  • Trascrizione e formalizzazione delle note vocali in report tecnici
  • Salvataggio e organizzazione dei tuoi report di cantiere
  • Generazione di PDF da condividere
  • Sincronizzazione dei dati tra dispositivi (solo con account)

I tuoi dati non vengono mai usati per pubblicità, profilazione o venduti a terze parti.

4. Servizi di terze parti

OpenAI (openai.com) Le registrazioni audio vengono inviate a OpenAI per la trascrizione (Whisper) e la formalizzazione del testo (GPT-4o). OpenAI elabora i dati secondo la propria Privacy Policy. I dati inviati via API non vengono usati da OpenAI per addestrare i modelli (secondo le loro API terms).
Supabase (supabase.com) Se crei un account, i tuoi report vengono memorizzati in un database Supabase ospitato in Europa (EU West). Supabase è conforme al GDPR. Consulta la loro Privacy Policy.

5. Dove vengono conservati i dati

  • Localmente: tutti i report sono salvati sul tuo dispositivo tramite AsyncStorage. Eliminando l'app elimini tutti i dati locali.
  • Cloud (con account): i report sincronizzati sono archiviati su server Supabase nella regione Europa (EU West-1), soggetti al GDPR.

6. I tuoi diritti

In qualsiasi momento puoi:

  • Accedere ai tuoi dati: sono tutti visibili nell'app
  • Esportare i tuoi report in formato PDF
  • Eliminare singoli report dall'app
  • Eliminare l'account e tutti i dati cloud: contattaci a bracco.ale@gmail.com
  • Usare l'app senza account: tutta la funzionalità principale è disponibile senza registrazione, con dati solo locali

7. Sicurezza

  • La comunicazione tra l'app e i server avviene tramite HTTPS
  • I token di autenticazione sono salvati in area sicura del dispositivo (SecureStore)
  • Il database Supabase ha Row Level Security (RLS) attiva: ogni utente vede solo i propri dati
  • L'Edge Function di trascrizione è protetta da chiave segreta

8. Minori

SiteVoice non è destinata a minori di 16 anni e non raccoglie consapevolmente dati di minorenni.

9. Modifiche a questa policy

Eventuali modifiche saranno pubblicate su questa pagina con la data di aggiornamento. Per modifiche sostanziali, gli utenti con account riceveranno una notifica via email.

10. Contatti
